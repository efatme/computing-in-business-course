Course Outline

Week Topic Hands-On
1 Introduction to Computing in Business, Digital Transformation Overview, Case Study: Amazon's use of IT in logistics Introduction to productivity tools (Google Workspace, MS Office Suite)
2 Hardware, Software & Business Systems, Basic Computer Architecture, Operating Systems & Enterprise Software
3 Spreadsheet Modeling for Business, Excel Fundamentals, Financial Modeling Basics, Profit/Loss Model Excel Formulas, Charts, Pivot Tables
4 Networking and Cloud Computing, Networking basics (LAN, WAN, VPN), Cloud services (SaaS, IaaS, PaaS), Lab: Set up a cloud-based workflow (Google Cloud or AWS), Case Study: Netflix cloud migration
5 Web Basics - HTML, HTML elements and page structure, Tables, links, forms for business reporting Structuring a simple Portfolio Website using HTML
6 Styling with CSS, CSS syntax and selectors, Inline vs internal vs external stylesheets, Layout design (flexbox/grid intro) Styling the Portfolio Website using CSS
7 E-commerce and Digital Marketing, Web technologies in business, SEO, email marketing, analytics, build a simple business landing website using no-code tools
8 Mid-Term Exam
9 Introduction to Programming with Python, Python syntax and IDE setup (Jupyter or VS Code), Variables, data types, input/output, Arithmetic operations, strings, lists, tuples, dictionaries, functions, modules, files and business decisions, for and while loops, Looping through lists and ranges, Nested loops and real-world business tasks Writing Python Scripts (business greeting, sales calculator)
10 Functions and Modularity, Writing and calling functions, Arguments, return values, scope Developing a practical small-scale application in Python
11 Data Structures - Lists and Tuples, Creating and manipulating lists, Tuples for immutable business data, Basic list operations, sorting, slicing, Storing business records using dictionaries, Sets for unique values (e.g., customer IDs)
12 File Handling and Exception Management, Reading from and writing to files, Handling missing files and input errors
13 Data Management and Numerical Analysis using Python Reading, Processing, Numerical Analysis, and Writing CSV and Excel Files using Pandas and Numpy in Python
14 Data Visualization, Descriptive vs predictive analytics, Data visualization using Excel and Python, KPI dashboards Creating an analytic dashboard, reports using Python
